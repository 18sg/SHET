SHET by 18sg

SHET House Event Tunnelling.

SHET is a simple asynchronous communication framework designed to make it easy to connect physical things together to produce useful behaviours. It was designed with home automation in mind, but has many other uses.

Dependencies

- [Python](http://www.python.org/) 2.x — it is known to work on at least 2.6 and 2.7. - [Twisted](http://twistedmatrix.com/)

Install

Download the code (you'll need [Git](http://git-scm.com/)): $ git clone git://github.com/18sg/SHET.git Install it: $ cd SHET $ sudo python2 setup.py install
